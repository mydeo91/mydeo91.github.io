[REACT-ERR] npm run start 시 Cannot find module 'typescript' 혹은 code ELIFECYCLE

2019-08-29
React

사이드 플젝을 하는데 나는 에러를 해결하는 시간만큼 아까운게 없다. 빨리 결과물을 내보고 싶은 마음이 큰데 이놈의 에러는 갑자기 뭐지??????? 어제까지는 잘 되던 친구가 갑자기 말을 안듣는다.

상황

mobs 의 decorator를 사용하기 위해서 eject를 했고, 그 결과 package.json 이 굉장히 더러워졌다. 그렇게 어제까지는 개발 서버를 잘 올려놓고 개발을 했다.

vscode 에서 이 decorator를 인식하고 꺼림직한 밑줄(warning)을 없애기 위해서 root 경로에 tsconfig.json 파일을 만들었고, 거기에

다음과 같이 파일 설정을 넣었다. 무슨 말인지도 잘 모르지만

문제

npm run start

저 놈의 ELIFECYCLE 은 정말 꼴보기 싫다. 참 친절하지 못한 코드다. 모두가 다른 해결방안을 제시하고 있기 때문에 그냥 npm 했는데 에러가 났고 설명하기 귀찮아서 저 코드를 던지는 것 같다. 결국 내가 스스로 찾아야한다.

시도

1. `npm cache clean --force && npm install`
2. `rm -rf node_modules && rm package-lock.json && npm install`
3. `npm install typescript`

해결

결국은 typescript 모듈이 없었기 때문에 생기는 문제였다. 나는 안쓰는데? 싶어서 확인해보니 vscode 설정으로 해두었던 tsconfig.json 파일이 유일하게 관련된 파일이었다.
rm tsconfig.json && npm install && npm run start

매우 잘 된다…….
혹여나 지친 몸을 이끌고 사이드 플젝을 하고 계신 분들에게 조그마한 도움이 되었으면 좋겠다.
